# Prompt & Profit — Errata & Feedback

This is the public errata tracker for **Prompt & Profit: The LLM Marketing Playbook**.

Found an unsourced statistic, a prompt that does not run, a broken citation, or a case study that has changed shape since publication? **[Open an erratum report](https://github.com/TTMAC/prompt-and-profit-errata/issues/new?template=errata.yml).**

A free GitHub account is required — GitHub handles sign-in, rate limiting, and spam detection, so reports stay readable and the queue stays clean.

## What happens after you submit

- **Acknowledgement** within seven days. Every report is read by the author.
- **Substantive fixes** ship within sixty days, in the next v1.x release.
- **Typos and broken links** ship between releases.
- Reader-reported corrections are credited by name on <https://promptprofits.tech/updates> when the fix ships, if you give a credit name in the report.

## Before you file

- The standards the Playbook holds itself to: <https://promptprofits.tech/standards>
- What gets flagged and how triage works: <https://promptprofits.tech/errata>

This repository holds issue templates only — no site or manuscript source.
